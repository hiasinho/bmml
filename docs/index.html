BMML

Business Model Markup Language

A YAML-based format for describing business models.
Based on Alexander Osterwalder's frameworks.

View on GitHub

Why BMML?

AI-Analyzable

Structured for machine parsing and analysis by AI agents

Version-Controllable

Git-friendly with meaningful diffs for tracking evolution

Validatable

Schema-enforced consistency and relationship checking

Human-Readable

Familiar YAML syntax for technical users

Business Model Canvas

Capture all 9 building blocks in a single .bmml file:

Customer Segments Value Propositions Channels Customer Relationships Revenue Streams Key Resources Key Activities Key Partnerships Cost Structure

Value Proposition Canvas

Deep-dive into customer-value fit with explicit relationships:

Jobs to be Done Pains Gains Products & Services Pain Relievers Gain Creators

Example

version: "1.0"

meta:
  name: "My Startup"
  tagline: "Solving real problems"
  portfolio: explore
  stage: validation

customer_segments:
  - id: cs-early-adopters
    name: "Early Adopters"
    jobs:
      - id: job-main
        type: functional
        description: "Get work done faster"
        importance: high

value_propositions:
  - id: vp-core
    name: "Core Product"
    products_services:
      - id: ps-app
        type: digital
        description: "Mobile application"

# ... fits, channels, revenue, costs
